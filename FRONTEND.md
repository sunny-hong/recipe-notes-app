# Typography Hierarchy
## Cormorant Garamond + DM Sans

---

## Cormorant Garamond (Titles & Hero Headings Only)

- Recipe title (e.g. "Grandma's Bolognese") — Cormorant Garamond, 600 weight, 28–32px
- Note title in editor — Cormorant Garamond, 600 weight, 24–28px
- Login page main heading — Cormorant Garamond, 600 weight, 32–36px
- App brand/logo text — Cormorant Garamond, 600 weight, 24px
- Section hero titles (if used) — Cormorant Garamond, 600 weight, 20–24px

---

## DM Sans (Everything Else)

### UI & Functional Text

- Toolbar / UI buttons — DM Sans, 400–500 weight, 14px
- Button labels (Create, Save, Delete) — DM Sans, 500 weight, 14–16px
- Form labels (e.g. "Email", "Password") — DM Sans, 500 weight, 14px
- Form inputs & text fields — DM Sans, 400 weight, 14–16px
- Search bar placeholder text — DM Sans, 400 weight, muted color, 14px

### Content & Reading

- Note body / recipe text — DM Sans, 400 weight, 15–16px
- Ingredient lists — DM Sans, 400 weight, 14–15px
- Instructions/steps — DM Sans, 400 weight, 15–16px
- List item titles (note list panel) — DM Sans, 500 weight, 15–16px
- List item preview text (snippet under title) — DM Sans, 400 weight, muted color, 13–14px

### Metadata & Secondary Info

- Meta info (e.g. "Last edited Monday · 4 min read") — DM Sans, 400 weight, muted color, 12px
- Tags (e.g. "pasta", "sunday") — DM Sans, 500 weight, small size (12px)
- Section labels (e.g. "Today", "Yesterday") — DM Sans, 500 weight, small size (12px), muted color
- Timestamps — DM Sans, 400 weight, muted color, 11–12px
- Helper text / error messages — DM Sans, 400 weight, 12px

---

## Weight Usage Summary

- **600 weight** = Cormorant only, titles & headings
- **500 weight** = DM Sans, labels, buttons, list titles, tags (emphasis without serif)
- **400 weight** = DM Sans, body text, inputs, metadata (default)

---

## The Rule of Thumb

**Cormorant Garamond is ONLY for:**
- Recipe/note titles
- Main headings (login, app brand)
- Hero section titles

**Everything else** — buttons, forms, body text, metadata, tags, UI labels — is **DM Sans**.

This keeps the interface functional and scannable while making titles feel warm, editorial, and distinctive.
